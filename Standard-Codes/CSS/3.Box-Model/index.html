Box model in CSS :

Box model is a concept which includes padding, margin, border, height, width.

Padding means the distance or seperation of a child element from its parent.

Margin decides the distance between siblings.

Height and width have usual meanings but the thing to be remembered is that the height/width will be not defined for whole div/element it will be for the content only.

Border also have usual meaning but it can be modified in sense of style, width, curving and many more.
