Selectors in HTML :

There are many types of selectors in CSS like : Element, universal, id, class selectors.

Through element selector we can select a specific element by using it's name.

Another selector we know is universal selector '*' ,through which we can select each element of out entire document.

Through Id selector we can select a perticular element with Id.

Through class selector we can select a perticular group elements with same class.
