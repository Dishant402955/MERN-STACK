Selectors in CSS :

There are many types of selectors in CSS like : Element, universal, id, class, descendent, attribute selectors.

Through element selector we can select a specific element by using it's name.

Through universal selector '*' ,through which we can select each element of out entire document.

Through Id selector we can select a perticular element with Id.

Through class selector we can select a perticular group elements with same class.

Through Descendant selector we can select a element inside a perticuler element.

Through attribute selector we can select elements based on their attribute values ,like input[type="text"] will select input elements of type text.


The order of importance in selectors is : id > class > element


Adjacent Sibling Combinator :

There are many types of combinator available like : +, >, ~

'+' is used to select a element which comes immediatly after it and are of same level, like " p + a " will select 'a' which comes immediatly after p .

'>' is used to select elements which are direct chidld of an element, like " p > a " will select all 'a' which are direct childs of 'p'.
