Grid-layout :

>> It helps to make 2-D layout .

>> By setting display to the grid it will make all the children grid items.

>> it is made up of row-lines and column-lines, which creates a grid-cell on the intersections .

>> grid-lines is used to define the distance between 2 lines.

>> grid-template is used to define the lines and the track sizing, with help of grid-template-rows, grid-template-columns, grid-template-areas .

>> grid-template-rows : repeat(count,1fr) will create 'count' rows each with track of 1fr of available width .

>> We can give gap with help of row-gap, column-gap, grid-gap .

>> grid-column is used to define an item's starting & ending position inside the column with syntax grid-column : start-col/end-col or grid-column : start-col/ span n .

>> grid-row is used to define an item's starting & ending position inside the row.
