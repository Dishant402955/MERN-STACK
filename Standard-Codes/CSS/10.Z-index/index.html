Z-index :

>> Z-index helps us to determine that which element will cover another while overlapping.

>> z-index : 0 is the default value of any element.

>> It's value can go from negetive infinity to positive infinity .

>> If the z-index is same for more than 1 property then it will follow the cascading property.

>> An important things is that z-index cannot be applied on the elements which have position : static or default.
