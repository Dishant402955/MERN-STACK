Selectors in CSS :

There are many types of selectors in CSS like : Element, universal, id, class, descendent, attribute, pseudo-class, pseudo-element selectors.

Element selector - we can select a specific element by using it's name.

Universal selector '*' - we can select each element of out entire document.

Id selector (#my-id) - we can select a perticular element with Id.

Class selector (.my-class) - we can select a perticular group elements with same class.

Descendant selector - we can select a element inside a particular element.

Attribute selector - we can select elements based on their attribute values ,like input[type="text"] will select input elements of type text.

Pseudo-class selectors - we can select element(s) on an special event like hover, active, checked, nth-of-type and many more...

Pseudo-element selectors - we can select a specific-part of an element for styling.



Specificity and cascading :

id >> class & attribute & pseudo-class >> element & pseudo-element.

Inline styling will have higher priority than these 3 , But if !important is given then it will have the highest priority than any other even more than inline.

Selector specificity by number : _ _ _ (if numbers come same than should use and consider cascading property.)



Adjacent Sibling Combinator :

There are many types of combinator available like : +, >, ~

'+' is used to select a element which comes immediatly after it and are of same level, like " p + a " will select 'a' which comes immediatly after p .

'>' is used to select elements which are direct child of an element, like " p > a " will select all 'a' which are direct childs of 'p'.
