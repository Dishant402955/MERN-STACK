Flex-box (Flexible box layout):

>> It is 1-D layout method for arranging items in rows or columns.

>> Important to create modern and responsive webpages.

>> flex-direction is used to define in which direction the container will fit the element : in row or row-reverse or column or column-reverse .

>> justify-content is used distribute space along main axis with values like flex-start/flex-end/center, space-around/space-evenly/space-between , baseline.

>> flex-wrap is used to set whether flex items are forced onto one line or can wrap onto multiple line which have values like nowrap, wrap, wrap-reverse .

>> align-items is used to distribute our items along cross axis with values like flex-start, flex-end, center.

>> align-content is used to distribute the space between & around content items along cross axis, with values like flex-start/flex-end/center , space-between/space-around/space-evenly, baseline .

>> align-self is used to align a item along the cross axis with values like flex-start, flex-end, center, baseline

>> flex-basis is used to set the initial main size of a flex item.

>> flex-grow is used to define the growing factor (when the container size will change).

>> max-width is used to define the upper-threshold of the width .

>> min-width is used to define the lower-threshold of the width .

>> flex-shrink is used to define the shrinking factor (when the container size will change).
