Forms in HTML :

>>> Action attribute is used to define that which action will be performed when a form is submitted.

>>> The main important element of form is 'input' and it have so many types.

>>> The types of input are text , password, date, number, checkbox, Radio button, datetime, email, color

>>> Placeholder is a attribute which is used to place some default text in the input field.

>>> Label is a element which is used to connect a label with a input field.

>>> We can directly put a submit, reset button in the form.

>>> Name is a attribute which is used to refer to the type of information we will be entering in the input field.
